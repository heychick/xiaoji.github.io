    ##1.环境准备

    192.168.1.10 kube-master

    192.168.1.100 registry 私有镜像仓库 安装docker-distribution

    192.168.1.11 node1 docker 安装docker
    192.168.1.12 node2 docker 安装docker
    192.168.1.13 node3 docker 安装docker

    ##2.创建kvm虚拟机
    qemu-img create -b .node_base.qcow2 -f qcow2 kube-master.img 30G
    cp -p zhj.xml kube-master.xml
    vim kube-master.xml
    virsh define kube-master
    virsh start kube-master

    ##3.安装完软件启动服务

    查看是否开机自启
    systemctl is-enabled docker

    仓库主机5000端口

    ##4.私有镜像仓库配置

    • 客户端:
      vim /etc/sysconfig/docker
    • docker仓库地址
      ADD_REGISTRY=’–add-registry 192.168.1.100:5000’
    • 允许非加密方式访问仓库
      INSECURE_REGISTRY=’–insecure-registry 192.168.1.100:5000’
    • 重启进程前要停止容器 docker stop $(docker ps -q)
      systemctl restart docker

    • 保存镜像
      docker save myos:latest -o myos.tar

    • 导入镜像
      docker load -i myos.tar

    • 为镜像创建标签

    • docker tag 镜像:标签 IP:5000/镜像:latese
      n1 docker tag myos:latest 192.168.1.100:5000/myos

    • 上传镜像

    • docker push IP:5000/镜像:latest
      n1 docker push 192.168.1.100:5000/myos

    • 远程启动镜像
      docker run -it 192.168.1.100:5000/myos

    • 查看私有镜像仓库中的 镜像名称
      curl http://192.168.1.100:5000/v2/_catalog

    • 查看某一仓库的标签
      curl http://192.168.1.100:5000/v2/myos/tags/list

    • 私有仓库数据存储目录
      /var/lib/registry/
